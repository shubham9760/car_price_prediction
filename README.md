# Car Price Prediction

A machine learning project for predicting car prices using advanced preprocessing, multiple model comparison, and comprehensive evaluation.

## Features

- Data ingestion and preprocessing
- Base model preparation
- Advanced model training with multiple algorithms
- Model evaluation and comparison
- Feature importance analysis
- REST API for predictions
- Docker containerization
- MLflow tracking for model versions

## Installation

```bash
pip install -e .
```

## Usage

```bash
python main.py
```
